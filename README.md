# Diagnóstico - Direção de Cena | João Paulo Coelho

App de análise de perfil e venda de consultoria estratégica para criadores.

**Fluxo:**
1. Quiz de diagnóstico (8 perguntas)
2. Cadastro do usuário
3. Tela de resultados + oferta dos pacotes
4. Briefing pré-sessão
5. Agendamento

**Stack:** React 19 + TypeScript + Vite + Tailwind + Hono + Cloudflare Workers

## Dev

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
npx wrangler deploy
```
